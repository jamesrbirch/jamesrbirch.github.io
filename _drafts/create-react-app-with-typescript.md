# Create a React application with TypeScript

To create a new react app start using typescript use:

```ps
npx create-react-app demo-app --template typescript
```

There are a number of other available:

-   [create-next-app](https://nextjs.org)
-   [create-snowpack-app](https://www.snowpack.dev)
-   [Nx](https://nx.dev)
-   [TSDX](https://tsdx.io)
